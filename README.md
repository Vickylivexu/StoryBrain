# StoryBrain - 故事化知识学习系统

基于 **concept-fable（概念寓言）** 和 **hv-analysis（纵横分析）** 的知识学习工具。通过故事帮助用户深度理解任意概念，输出可直接导入 Obsidian 等笔记工具。

## 核心能力

1. **📖 概念寓言** - 用一个引人入胜的故事诠释核心概念
2. **🔍 纵横分析** - 纵向追溯起源发展 + 横向对比相关概念
3. **🏷️ 知识分类** - 学科分类、核心概念、应用领域、学习路径、关键词标签
4. **📥 Markdown 导出** - 一键导出为 Obsidian 兼容格式
5. **✅ 质量验证** - 14 项检查规则保证输出质量

## 使用方式

在 TRAE 中调用 `story-brain` skill：

```
用户：我想深入了解一下区块链

助手：[调用 story-brain skill]
  → 生成寓言故事 + 纵横分析 + 知识卡片 + 分类标签
  → 质量验证通过后交付
  → 用户可要求导出 Markdown 格式
```

### 触发条件

- "我想深入了解一下 XXX"
- "帮我分析一下 XXX"
- "用故事解释 XXX"
- "我想了解 XXX"
- "帮我研究 XXX"
- "把 XXX 整理成笔记"
- "帮我分析大学专业 XXX"

## 扩展主题

在 `data/` 目录新增 `{主题名}.json` 文件，遵循 SKILL.md 中的 JSON 格式规范，系统自动识别。

## 技术栈

- **Concept Fable** - 用隐喻故事解释抽象概念
- **HV-Analysis** - 纵向 + 横向深度分析
- **Obsidian 兼容** - YAML frontmatter + 双链格式
